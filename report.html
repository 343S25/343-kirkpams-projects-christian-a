Milestone Report

  1. Github Repo: Github

  2. Deployed Site: Website

  3. Media queries are implemented on each page typically switching from a grid layout to flex column layout

  4. The website contains 4 pages. The home, nutrition, exercise, and bio page

  5. The site includes an identical header on each page. A header.css exists to give a uniform look across pages. It also changes when on smaller screens

  6. The site includes nav links in the header as well as in the grids on the homepage that bring you to their respective pages.

  7. A heart image is used next to the site header title with a descriptive alt attribute.

  8. All pages were validated without warnings according to Nu HTML Checker and WAVE Accessibility

  9. Headings are using Helvetica while paragraphs and other smaller text are using Georgia

  10. A form in the nutrition page was included so that the user can enter the nutrition facts of their food

  11. The site includes a more resources footer that includes links to external sites

  12. I used the USDA API to get food data that will be able to be stored in the nutrition journal and the World Time API to server as a clock at the top of the website
